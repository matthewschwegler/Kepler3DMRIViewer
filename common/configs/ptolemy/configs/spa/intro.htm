SPA: Scientific Process Automation

The Scientific Process Automation project enables scientists to easily create exploratory data flows (scientific workflows) from local and distributed components (e.g. web services).

Below is a sampling of workflows powered by SPA.

Workflow Description
Promotor Identification Workflow A production workflow that generates a list of promoter binding sites for each of a given set gene ID numbers.

SPA is a collaboration between Lawrence Livermore National Laboratory, San Diego Supercomputer Center at UCSD, and North Carolina State University. It is based upon the Ptolemy II heterogenous modeling and design framework. It is also an active participant in the Kepler project.
